# Classzle v1.2.3 릴리즈 노트

**배포일:** 2025-12-27
**빌드 환경:** Windows (Electron)

## 🚀 주요 개선 사항

### 1. 알고리즘 및 균형 배정 고도화
- **생활지도 점수/유형 정밀 균형 (Behavior Balance)**:
  - 행동/정서 유형(리더, 행동, 정서) 및 각 점수대(-1, -2, -3 등)별 학생 수가 **모든 반에 균등하게** 배정되도록 강력한 균형 알고리즘을 적용했습니다.
- **남녀/성적 후처리 보정 (Post-Processing)**:
  - '신규 배정' 및 '현재 배정 수정(Optimize)' 모드 모두에 대하여, 1차 배정 후 **성비와 성적 평균**을 다시 한번 미세 조정하는 후처리 단계를 추가하여 최적의 균형을 달성합니다.

### 2. 배정 리포트(Report) 전면 개편
- **디자인 및 레이아웃 표준화**:
  - 리포트 내 모든 탭(이행 현황, 이동 명단, 변경 이력)의 폰트, 배지(Badge) 스타일, 아이콘을 시스템 표준으로 통일하여 가독성과 전문성을 크게 높였습니다.
  - 출력(인쇄) 시 **'그룹 및 특수 배정 현황'이 상단에 배치**되도록 레이아웃을 최적화하고, 페이지 넘김이 자연스럽도록 개선했습니다.
- **데이터 표현 개선**:
  - **이동 명단**: 중복되던 '이전 정보' 열을 제거하고, '최초 배정 반 → 최종 배정 반' 흐름을 한눈에 볼 수 있도록 간소화했습니다. 정렬 기준도 '최초 반/번호' 순으로 변경하여 확인이 쉬워졌습니다.
  - **이행 현황**: '같은 반 희망/피해야 할 관계' 목록에서 중복 표시를 제거하고, 이전 학년 반/번호 순으로 정렬하여 담임 선생님들의 확인 편의성을 높였습니다.
  - **변경 누적 이력**: 명칭을 '변경 상세 이력'에서 **'변경 누적 이력'**으로 변경하고, 데이터가 시간순(과거→최신)으로 정확히 쌓이도록 로직을 검증했습니다.

### 3. 사용자 인터페이스(UI) 편의성
- **관계 설정 아이콘 직관성 강화**:
  - **같은 반 희망 (Keep)**: 기존의 헷갈리던 화살표 대신 **'악수하는 손(Handshake 🤝)'** 아이콘을 적용하여 '협력/함께함'의 의미를 명확히 했습니다.
  - **피해야 할 관계 (Avoid)**: **'붉은색 양방향 화살표(ArrowsLeftRight ↔)'**로 통일하여 '분리/경고'의 의미를 강조했습니다.

### 4. 버그 수정 및 안정성
- **데이터 가져오기 초기화 오류 수정**:
  - 새로운 엑셀 파일을 업로드할 때, 기존에 설정했던 **'분산 배정 그룹'** 정보가 남아있던 문제를 해결했습니다.
  - 이제 파일을 새로 불러오면 학생 명단, 그룹 설정, 변경 이력이 모두 깔끔하게 초기화됩니다.
