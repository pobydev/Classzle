# Classzle v1.2.4 릴리즈 노트

**배포일:** 2025. 12. 31.

클래즐(Classzle) v1.2.4 업데이트에서는 데스크톱 앱(Electron)의 사용성을 저해하던 입력 포커스 문제를 해결하고 인터랙션의 안정성을 높였습니다.

### 🛠 주요 수정 및 개선 사항

#### 1. 일렉트론 입력 포커스 먹통 버그 수정
*   **문제 현상:** 프로젝트를 불러온 직후 검색창이나 학생 정보 수정창을 클릭해도 커서가 생기지 않아 입력을 할 수 없던 현상
*   **원인 해결:** 기존 동기식(Synchronous) 브라우저 대화창(`confirm`)을 일렉트론 네이티브 비동기 대화창으로 교체하여 렌더러 프로세스의 차단을 방지했습니다.
*   **안정성 강화:** 드롭다운 메뉴가 닫히는 시점과 대화창이 뜨는 시점의 충돌을 방지하기 위해 미세한 지연 처리를 추가하고, 작업 완료 후 메인 윈도우로 명시적인 포커스 복구 로직을 적용했습니다.

#### 2. 사용자 인터랙션 안정성 강화
*   **네이티브 Confirm 다이얼로그 적용:** '작업 내용 불러오기', '프로젝트 삭제', '데이터 전체 초기화' 등 주요 확인 절차에 일렉트론 순정 다이얼로그를 적용하여 OS 수준에서의 창 관리 안정성을 확보했습니다.

#### 3. 문서 및 버전 업데이트
*   프로젝트 버전 정보를 v1.2.4로 상향하고 관련 PRD 문서를 업데이트했습니다.
