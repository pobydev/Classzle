# Classzle v1.1.0 Release Notes

## 🎯 반편성 균형 알고리즘 개선

### ✨ 주요 변경사항

#### 1. 신규 배정 균형 강화
- **유형+점수별 균형 개선**: 행동형 -2와 행동형 -1 학생이 각 반에 더 고르게 배정됩니다.
- **교환 조건 완화**: 기존에 교환이 되지 않던 상황에서도 균형을 맞추기 위한 교환이 발생합니다.
- 예: 행동형 -2 학생이 1반에 3명, 2반에 0명인 경우 → 자동으로 분산 배정

#### 2. 배정 수정 모드 개선 (기배정 모드)
- **심각한 불균형 자동 교정**: 기존 배정의 틀은 유지하면서, 심각한 불균형만 교정합니다.
- 대상: 행동형/정서형 **-2, -3점** 학생 (가장 관리가 필요한 학생)
- 조건: 반별 차이가 **2명 이상**일 때만 교환
- 결과: 기존 배정의 80-90% 이상 유지, 심각한 불균형만 해결

### 🔧 기술적 변경사항
- `balanceByTypeAndScore` 함수 교환 조건 완화
- `balanceByTypeAndScoreStrict` 함수 신규 추가 (기배정 모드용)
- 코드 리팩토링: 타입 마이그레이션, 상수 중앙화, 컴포넌트 분리
