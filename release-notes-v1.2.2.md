# Classzle v1.2.2 릴리즈 노트

이번 v1.2.2 버전은 기존 v1.2.1 대비 **'재배정(Optimize)' 기능의 강력한 성능 향상**에 집중했습니다.

## 🚀 주요 변경 사항 (v1.2.1 대비)

### 1. '현재 배정 수정' 엔진의 비약적 진화
기존 버전에서 **현재 배정 수정(Optimize)** 시 조건 해결이 다소 소극적이었던 점을 개선하여, **'공격적 문제 해결(Aggressive Solver)'** 엔진을 탑재했습니다.

- **강력한 제약 조건 해결**: '피해야 할 관계', '같은 반 희망' 등, 사용자가 수동으로 조작하다가 깨진 복잡한 제약 조건들을 시스템이 훨씬 더 적극적으로 개입하여 해결해 냅니다.
- **엄격한 인원/성적 균형 강제**: 재배정 과정에서 흐트러질 수 있는 반별 인원수와 성적 평균을 오차 범위(1명) 내로 강력하게 맞춥니다.
- **사용자 의도 존중**: 사용자가 배치해 둔 틀을 최대한 유지하면서도, 위반 사항이 발생한 부분만 핀셋처럼 찾아내어 수정합니다.
